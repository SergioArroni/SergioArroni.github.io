# SergioArroni.github.io

Personal portfolio & CV — **Sergio Arroni del Riego**

AI Architect & Researcher | Industrial PhD Student in Intelligent Systems

## Tech Stack

- **Framework**: [Astro](https://astro.build/) (Static Site Generation)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Interactive Graph**: [D3.js](https://d3js.org/) (Force-directed skill graph)
- **i18n**: Bilingual (ES / EN)
- **Deploy**: GitHub Pages via GitHub Actions

## Development

```bash
npm install
npm run dev      # Dev server
npm run build    # Production build
npm run preview  # Preview build
```

## Design: "Neural Garden"

A dark tech aesthetic with organic nature accents — bioluminescent particles, vine-like separators, and a force-directed graph where skills connect like neural pathways through a digital forest.
